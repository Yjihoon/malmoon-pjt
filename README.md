# 🧠 말문 (Malmoon) - WebRTC 기반 언어치료 플랫폼

## 📌 프로젝트 개요

**말문(Malmoon)**은 언어지연 및 학습 부진 아동을 위한  
**실시간 WebRTC 언어치료 플랫폼**입니다.  
언어재활사가 직접 세션을 구성하고, 아동은 AAC 인터페이스 및 디지털 동화를 통해 효과적으로 소통하며 치료에 참여할 수 있도록 설계되었습니다.

> **주요 키워드**: WebRTC, AAC, STT, 언어치료, 실시간 세션, 자동 리포트

---

## ✨ 주요 기능 (예정)

- ✅ WebRTC 기반 1:1 실시간 언어치료 세션
- ✅ 치료사가 직접 설정 가능한 AAC 이모지 도구
- ✅ 동화 기반 문장 제공 및 대상자 STT 분석
- ✅ 집중력 필터 및 상호작용 지원 기능
- ✅ 치료 후 자동 리포트 생성 및 이력 관리
- ✅ 대상자-치료사 스케줄 매칭 및 간편 진단
